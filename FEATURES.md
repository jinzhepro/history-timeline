# 人物关系图谱功能说明

## 功能概述

人物关系图谱功能已成功添加到中国历史时间线应用中。该功能使用 ECharts 可视化展示各朝代重要历史人物之间的关系网络。

## 已实现的功能

### 1. 数据结构扩展
- 在 `/data/dynasties.js` 中添加了 `historicalFigures` 数组
- 包含 70+ 位重要历史人物的详细信息
- 每个人物包含：姓名、字号、头衔、在位时间、描述、社会关系

### 2. 组件创建

#### FamilyTree.js - 人物关系图谱组件
- 使用 ECharts 力导向图布局
- 支持拖拽、缩放、漫游
- 点击节点显示详情
- 悬停显示详细信息

#### PersonCard.js - 人物详情卡片
- 展示人物完整信息
- 显示社会关系列表
- 带图标的关系类型标识

### 3. 集成到详情抽屉
- 在 DynastyDrawer 中添加了"查看人物关系图谱"按钮
- 可展开/收起关系图谱区域
- 点击图谱节点自动显示人物详情

## 使用方法

### 查看人物关系图谱
1. 点击任意朝代卡片打开详情抽屉
2. 点击"查看人物关系图谱"按钮
3. 图谱将显示该朝代的人物关系网络

### 交互操作
- **拖拽节点**: 调整人物位置
- **滚轮缩放**: 放大/缩小查看细节
- **点击节点**: 选择人物并显示详情
- **悬停**: 显示人物简介 tooltip

### 关闭图谱
- 再次点击"隐藏人物关系图谱"按钮
- 或关闭朝代详情抽屉

## 调试信息

组件已添加详细的控制台日志:
- `[FamilyTree] Dynasty: {id}, Found {count} figures` - 显示找到的人物
- `[FamilyTree] Figure map: [...]` - 显示人物 ID 映射
- `[FamilyTree] Created node: {id} {name}` - 显示创建的节点
- `[FamilyTree] Created link: {source} -> {target}` - 显示创建的关系
- `[FamilyTree] Final graph data: {nodes, links}` - 显示最终数据

打开浏览器控制台可以看到这些日志，帮助排查问题。

## 故障排除

### 图谱显示空白
1. 打开浏览器控制台 (F12)
2. 查看是否有错误信息
3. 检查 `[FamilyTree]` 开头的日志
4. 确认该朝代有人物数据

### 节点不显示
1. 检查人物数据中的 `dynasty` 字段是否匹配
2. 确认关系的目标人物也存在于同一朝代
3. 尝试拖拽或缩放查看是否有节点

### 样式异常
1. 清除浏览器缓存
2. 确保 globals.css 正确加载
3. 检查 Tailwind CSS 配置
